# Functions

## Circle funcions

- Area

```
>>> area(2)
12.566370614359172
```

- Perimeter

```
>>> perimeter(4)
25.132741228718345
```

## Square functions

- Area

```
>>> area(2)
4
```

- Perimeter

```
>>> perimeter(2)
8
```
